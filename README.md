# Fine-Tuning PoCs

This repository was created to help understand how fine-tuning works.
It is meant to be a learning space where I can explore and document practical examples of fine-tuning different models.

## Purpose

The goal of this project is to build small proof-of-concept examples that show how fine-tuning is done in practice, what changes are required, and how results can be evaluated.

## What this repository will contain

- Simple fine-tuning examples
- Small training and evaluation scripts
- Model inference examples
- Notes and experiments as the repository grows

This repo will serve as a reference for learning and experimenting with fine-tuning workflows.
